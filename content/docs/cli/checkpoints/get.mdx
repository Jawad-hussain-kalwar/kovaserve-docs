title: "`kova checkpoints get`"
description: Fetch one Checkpoint by ID. Renders the bound ContextManifest, the typed StateHandles, and the compatibility vector.
---

Fetch one [Checkpoint](/docs/concepts/recovery/checkpoints) by ID with its bound [ContextManifest](/docs/concepts/context/context-manifests), typed [StateHandles](/docs/concepts/recovery/state-handles), and compatibility vector.

## Synopsis

```bash
kova checkpoints get <CHECKPOINT_ID> [OPTIONS]
```

## Options

| Option | Meaning |
|---|---|
| `--expand <path>` | Hydrate nested resources. Valid: `context_manifest`, `context_manifest.segments`. |
| `--json` | Emit the Checkpoint as one JSON document. |

## Examples

```bash
kova checkpoints get 01J8X9Y2Z3K4M5N6P7Q8R9S0T7

kova checkpoints get 01J... --expand context_manifest.segments

kova checkpoints get 01J... --json | jq '.state_refs[] | {type: .ref_type, location: .location}'
```
