---
title: "`kova context segments`"
description: Fetch individual ContextSegments — the typed fragments that compose a ContextManifest.
---

Fetch individual [ContextSegments](/docs/concepts/context/context-segments). Segments are immutable artifacts of manifest assembly; this command is read-only.

## Subcommand

| Subcommand | What it does |
|---|---|
| `kova context segments get <ID>` | Fetch one segment with its body (inline) or artifact reference (offloaded). |

To list segments belonging to a manifest, use `kova context manifests get <ID> --expand segments` — segments are most usefully addressed through the parent manifest.

## Options

| Option | Meaning |
|---|---|
| `--json` | Emit the segment as one JSON document. |
| `--show-body` | For inline segments, print the full body. Default truncates at 2 KB. |

## Examples

```bash
kova context segments get 01J8X9Y2Z3K4M5N6P7Q8R9SSG12

# Full body for inline segment
kova context segments get 01J... --show-body

# JSON
kova context segments get 01J... --json | jq '{type: .context_type, class: .memory_class}'
```
