---
title: Get a Project
description: GET /v1/ks/projects/{id} ŌĆö fetch a Project with its resolved PolicyScope, default context-share scope, and lifecycle status.
---

`GET /v1/ks/projects/{id}` returns a single [Project](/docs/concepts/tenancy/projects). The response carries the parent Tenant, the bound [PolicyScope](/docs/concepts/policy/policy-scopes) override (if any), the default cache-share scope for Runs in this Project, and the lifecycle status.

The Project is resolved against the API key's Tenant. Cross-tenant fetches return `404 Not Found` ŌĆö the same response as a genuinely missing Project ŌĆö so the surface leaks no information about Projects the caller cannot see.

## Endpoint

```
GET /v1/ks/projects/{project_id}
```

| Property | Value |
|---|---|
| Auth | `Authorization: Bearer ks_...` ŌĆö see [Authentication](/docs/api/using-the-api/authentication). |
| Path parameter | `project_id` ŌĆö bare 26-character Crockford ULID. |
| Errors | Native envelope ŌĆö see [Errors](/docs/api/using-the-api/errors). |

## Response ŌĆö `200 OK`

```json
{
  "id": "01J8X9Y2Z3K4M5N6P7Q8R9PRJ1",
  "tenant_id": "01J8X9Y2Z3K4M5N6P7Q8R9ORG1",
  "name": "coding-agent",
  "status": "ACTIVE",
  "policy_scope_id": "01J8X9Y2Z3K4M5N6P7Q8R9P0K1",
  "default_context_share_scope": "project",
  "default_compaction_policy": null,
  "created_at": "2026-04-30T12:00:00Z"
}
```

| Field | Type | Meaning |
|---|---|---|
| `id` | string | Project ID. |
| `tenant_id` | string | Owning Tenant. Immutable. |
| `name` | string | Human-readable identifier, unique within the Tenant. |
| `status` | string | `ACTIVE` or `ARCHIVED`. Archived Projects reject new admission with `403 Forbidden` and `error.code = "project_archived"`; existing rows remain queryable. |
| `policy_scope_id` | string \| null | Project-level PolicyScope override, or `null` when the Project inherits the Tenant default. |
| `default_context_share_scope` | string | `none` / `project` / `tenant`. The default cache and manifest reuse boundary applied to Runs in this Project. |
| `default_compaction_policy` | object \| null | Default compaction strategy hint for Runs that do not declare their own. |
| `created_at` | string | ISO-8601 timestamp with `Z` suffix. |

The `default_context_share_scope` is the Project-level setting; the resolved scope on each Run can be narrowed further by the [PolicyScope](/docs/concepts/policy/policy-scopes#context_share_scope) overlay applied at admission time.

## Status codes

| Status | When |
|---|---|
| `200 OK` | The Project exists and is visible to the caller. |
| `401 Unauthorized` | Missing or invalid bearer token. Codes: `missing_token`, `invalid_token`, `token_revoked`. |
| `404 Not Found` | The Project does not exist *or* belongs to another Tenant. Code: `project_not_found`. |

## Examples

```bash
curl https://api.kovaserve.com/v1/ks/projects/01J8X9Y2Z3K4M5N6P7Q8R9PRJ1 \
  -H "Authorization: Bearer ks_live_a7f3c2b8d4e1f0925a1b6c3d4e5f6a7b"
```

```python
from kovaserve import KovaClient

kova = KovaClient(api_key="ks_live_a7f3c2b8d4e1f0925a1b6c3d4e5f6a7b")

project = kova.projects.get("01J8X9Y2Z3K4M5N6P7Q8R9PRJ1")

print(f"{project.name}  status={project.status}")
print(f"  share_scope: {project.default_context_share_scope}")
if project.policy_scope_id:
    print(f"  policy:      {project.policy_scope_id}")
```
