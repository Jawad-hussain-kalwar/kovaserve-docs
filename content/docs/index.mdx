**KovaServe is the Agentic Execution Cloud for neoclouds and private AI environments** — the execution control plane between open inference substrate (vLLM / llm-d) and agent runtimes (LangGraph, Temporal, custom).

The unit of value is a **completed agent run**, not a token or a request.

## Why KovaServe exists

The first generation of AI infrastructure exposed models and inference endpoints. Production agents require more than model calls. They execute tasks over time. They use tools, maintain context, accumulate state, consume budget, fail, recover, and create operational risk.

KovaServe is the execution cloud required to make those agents production-ready — durable runs, checkpoint recovery, policy governance, budget enforcement, and cost visibility, all running across private, sovereign, neocloud, or enterprise-controlled infrastructure.
